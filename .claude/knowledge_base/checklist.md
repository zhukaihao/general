# 覆盖度检查清单

## 牌类型参考

> 语义筛选时用于判断牌类型归属，如「非基本牌」包含哪些牌、「伤害牌」包含哪些牌。

### 基本牌

杀、闪、桃、酒

### 锦囊牌

#### 普通锦囊

决斗、南蛮入侵、万箭齐发、桃园结义、无中生有、过河拆桥、顺手牵羊、借刀杀人、无懈可击、火攻、铁索连环

#### 延时锦囊

闪电、乐不思蜀、兵粮寸断

### 装备牌

武器、防具、+1马、-1马、宝物

### 伤害牌

杀（含雷杀、火杀）、决斗、南蛮入侵、万箭齐发、火攻

## 通用测试点

### 通用必测项

> 以下测试点必须挂载到每个需要玩家操作的短语上，不限于「可/可以」。

| 短语特征 | 挂载测试点 |
|----------|-----------|
| 包含「可/可以」的主动发动短语 | 「可以取消发动」「超时默认不发动」 |
| 不含「可/可以」但需要玩家选择的短语（如选择目标、选择牌、选择选项） | 「不能取消发动」「超时随机选择/超时默认选择」 |

### 通用测试点 -检查清单

| 类别| 测试点 |
|--------|--------|
| `效果_伤害`  | 「体力流失不触发」 「一次多点伤害的情况」 |
| `观看牌堆顶` | 「看牌的展示顺序」 「剩余牌是否按顺序返回牌堆顶」 |
| `选项分支` | 「选过的选项是否灰置」 「没有可选项时的情况」「只有一个选项的超时情况」 |
| `出牌阶段` | 「额外出牌阶段」 「非空闲时机点不能点击」 |
| `每回合` | 「其他角色回合结束时能重置」 「自己回合结束时能重置」「每轮开始时或游戏开始时不算在回合内」 |
| `效果_持续效果` | 「持续效果在技能被封的情况下生效」 「持续效果在角色阵亡的情况下生效」 |
| `触发_失去牌`| 「使用装备应该不算失去牌」 「使用装备应该算失去手牌」「一次失去多张的情况」 |
| `效果_无次数限制` | 「先出牌再无次数限制应该可以出」 「先无次数限制再出牌应该可以出」「酒应该可以无次数限制使用」「触发后技能被封的情况」 |
| `效果_无距离限制` | 「基本和锦囊应该都可以无距离限制」 「触发后技能被封的情况」 |
| `效果_不能使用`  | 「转化牌的情况」 「虚拟牌的情况」 「区分使用和打出」「铁索连环重铸的情况」「装备区的牌是否能使用」|
| `额外选择` | 「额外选择和正常选择的执行顺序」 「额外选择方式:1+1/方天」 「额外选择是否视为多目标」「额外选择是否需要合法」「额外选择用的牌是否和原牌同属性」「濒死情况指定」|
| `额外执行` | 「额外执行和正常选择的执行顺序」 「额外执行是否计算使用多张牌」 「额外执行是否计算成为多次目标」「额外执行用的牌是否原牌同属性」|
| `手牌颜色`  | 「颜色变化的情况」 「无色牌也是颜色」 |
| `效果_观看牌堆顶` | 「观看牌堆顶的展示顺序」 「观看后牌是否需要放回」「其他角色是否能看」 |
| `效果_放回牌堆` | 「放回的顺序是否为左边第一张为顶」 「客户端调整是否能流畅地调整」 |
| `效果_废除` | 「应该不能向废除的区域中移动牌」 |
| `类型_限定技` | 「限定技灯泡显示」 「限定技发动大图和发动后显示」|
| `类型_锁定技` | 「锁定技标识」「锁定技应该不可取消发动」 「锁定技应该不会被禁用」|
| `效果_转化牌` | 「选择被强转的牌转化」 「没有桃时，如果可以转化装备也要能使用」「转化牌的杀是否能选属性杀」「如果牌本身有特效，转化后是否有特效」 「手上没有对应牌时，如果能转化出也应该要询问」「转化火攻是否能对自己使用」|
| `效果_虚拟牌` |  「没有桃时应该可以视为使用桃」「没有无懈时应该也会询问是否发动无懈」「不能使用手牌时，应该可以视为使用」「若可以视为使用锦囊牌，火攻应该可以选择自己」「视为使用的距离和次数限制」「场上有卢奕选择目标的情况」|
| `触发_成为目标` | 「牌被无懈了也要能触发效果」「濒死用桃指定也会触发」 |
| `判断_最少/多` | 「有多个相同的情况」 |
| `移动` | 「移动牌不能往废除的区域移动」「移动牌不能往已经有牌的区域移动」「是否可以选择判定牌」「特殊装备被移动了应该销毁」 |
| `效果_不能响应` | 「自己能否响应」 |
| `效果_交给` | 「应该能触发谋黄权/伊籍的技能」 |
| `效果_弃置` | 「自己手牌无法弃置的情况/董白」「其他角色的牌无法弃置的情况/李丰」 「发动技能无法弃牌/谋庞统」 |
| `选择目标` | 「不能选择已阵亡角色」「选择0号位发动的情况不异常」「判断是否应能选择自己」 |
| `时机_游戏开始时` | 「应该在分发手牌时机之后」 「应该不计算入第一个玩家的回合内」「和乐系列武将同一个时机点」|
| `时机_每轮开始/结束` | 「每轮开始时不计入第一个玩家回合之内」「每轮结束时不计入最后一个玩家回合之内」 |
| `时机_分发手牌` | 「时机在乐系列武将之前」 |
| `效果_标记` | 「获得标记时数量正确显示」「移动标记到其他角色处后原角色标记减少」「标记数量为0时相关选项灰置」「标记在角色阵亡后消失」「客户端显示标记icon和数量」 |
| `牌名` | 「杀、火杀、雷杀是相同牌名」「五个字的情况」|
